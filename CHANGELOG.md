# YJYX Web Client Changelog

## 2017.6.13, Version 1.1.0(Test)

### New Feature
1. 个人中心新增个人信息展示、更改班级、修改密码和查看我的订单
2. 学生端任务列表新增搜索条件(关键字)和过滤条件(学科和任务状态)
3. 薄弱点展示及其知识讲解视频观看
4. 错题库
5. 亿教同步课堂
